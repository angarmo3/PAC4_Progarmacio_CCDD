Aquest projecte serveix per a contestar els exercicis proposats
a la PAC4 de l'assignatura Programació per a la Ciència de Dades.

Consta d'una sèrie de funcions, comandes de test, exemples i altres
scripts i informació d'interés recollides en:

- README.txt: informació sobre el projete
- INSTALL.txt: instal·lació de paquets a requirements.
- CONTRIBUTING.txt: fitxer amb el nom del contribuient.
- LICENSE.txt: fitxer amb la llicència (gratuïta i lliure).
- requirements.txt: amb els paquets necessaris per a executar el projecte.
- funcions_exercici_1_PAC4.py: fitxer pla de Python amb les funcions
	per a l'exercici 1.
- funcions_exercici_2_PAC4.py: fitxer pla de Python amb les funcions
	per a l'exercici 2.
- funcions_exercici_3_PAC4.py: fitxer pla de Python amb les funcions
	per a l'exercici 3.
- main.py: fitxer pla de Python per a executar les solucions de la PAC4.
- test: carpeta amb els scripts per a testar els funcions definides.
